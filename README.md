## 🚀 What is Salon Auto Recall CRM?

Salon Auto Recall CRM is an open-source automation framework designed for
appointment-based businesses.

It helps small businesses transform customer history data into actionable retention workflows without requiring expensive CRM SaaS platforms.

Although originally designed for beauty salons, the architecture is built to be extended for:

- 💇 Beauty salons
- 🏥 Clinics
- 🧘 Studios
- 🏪 Local service businesses


## ✨ Why this project?

Many small businesses have customer data,
but lack simple tools to answer:

- Who should we contact today?
- Which customers are likely to leave?
- What message should we send?
- How can marketing content be generated automatically?


This project provides a lightweight,
self-hosted foundation for customer retention automation.

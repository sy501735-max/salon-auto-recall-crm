# 기여 가이드 (Contributing Guide)

이 프로젝트에 기여해주셔서 감사합니다! 🎉

## 기여 방법

### 버그 리포트
- [Issues](../../issues)에서 버그를 신고해주세요
- 재현 방법, 예상 결과, 실제 결과를 함께 작성해주세요

### 기능 제안
- 아이디어가 있으시면 Issues에 `[Feature Request]` 태그로 남겨주세요

### 코드 기여
1. 이 저장소를 Fork합니다
2. 기능 브랜치를 만듭니다: `git checkout -b feature/기능이름`
3. 변경사항을 커밋합니다: `git commit -m 'feat: 기능 설명'`
4. 브랜치에 Push합니다: `git push origin feature/기능이름`
5. Pull Request를 열어주세요

## 코드 스타일
- Python 3.9+ 호환
- 함수와 클래스에 한국어 docstring 권장
- 새 기능에는 테스트 코드 추가 권장
